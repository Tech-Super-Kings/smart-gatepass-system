# Smart Digital GatePass System

## 🚀 Project Overview
A fast, secure, and contactless visitor management system using QR and OTP.

## 👥 Team
- Ashwin Kumar A – Frontend
- Gokul S – Backend
- Vigneshwaran V – Security & System Design
- Piyush Kumar – Database & Integration

## 🛠 Tech Stack
- Frontend: Next.js
- Backend: Node.js + Express
- Database: MongoDB
- Auth: Firebase / Twilio
- Hosting: Vercel / Render

## ⚙️ Features
- QR-based entry
- OTP verification
- Real-time tracking
- Pre-registration
